# wechat-charts
微信小程序图表工具，charts for WeChat small app

基于canvas绘制，体积小巧

# 支持图表类型
- 饼图   `pie`
- 圆环图 `ring`
- 线图   `line`
- 柱状图 `column`
- 区域图 `area`
- 雷达图 `radar`

代码分析 [Here](https://segmentfault.com/a/1190000007649376)

# 如何使用
直接引用编译好的文件 `dist/wechatcharts.js` 或者 `dist/wechatcharts-min.js`
